# **App Name**: StudentFlow

## Core Features:

- Admin Authentication: Secure login system for authorized school staff with password encryption.
- Student Record Management: Ability to add new, view, edit, and delete individual student records and their details.
- Student Photo Upload: Functionality to upload and associate a passport photograph with each student's profile.
- Student Search & Filtering: Efficiently search for students by name or admission number, and filter records by class or student status.
- Dashboard Overview: A centralized dashboard displaying key statistics and summaries like total students and active students for quick insights.

## Style Guidelines:

- Primary color: A clear, professional medium blue (#225C9E) evoking trust and clarity, appropriate for an administrative system.
- Background color: A very light, almost off-white blue (#EBF2F7) to provide a clean and calm canvas, preventing eye strain during long use.
- Accent color: A brighter, contrasting purple-violet (#7C7CEF) for interactive elements, highlights, and important calls to action.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean lines and excellent legibility, ensuring clarity for data-heavy displays.
- Clean, structured layouts with clear divisions and consistent spacing to improve data readability and ease of navigation for administrative tasks.
- Utilize minimalist, line-based icons that are universally recognized for actions like edit, delete, add, and search to maintain a professional look.
